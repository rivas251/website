Información sociodemográfica por entidad federativa

Esta aplicación web ha sido desarrollada para la materia de Taller de Big Data en la Nube. El código base ha sido tomado de la página web "Simplemaps". El mapa es generado por una combinación de archivos html5 y java script que han sido modificados para mostrar la información deseada.

La aplicación permite visualizar información demográfica tomada de la Encuesta Intercensal 2015. El usuario puede dar click en cualquier estado y una vez que se hace zoom, puede dar un segundo click para descargar tabulados con información más detallada (población por municipio y tamaño de localidad dividida por sexo, edad y condición de registro de nacimiento..

Fuente: Instituto Nacional de Estadística y Geografía (2016). Encuesta Intercensal 2015 [Tabulados básicos]..
